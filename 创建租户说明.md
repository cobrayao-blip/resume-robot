# 创建租户说明

## 租户管理员账户创建逻辑

### 方式一：创建租户时同时创建管理员账户（推荐）

在创建租户时，可以**可选地**填写租户管理员信息：

1. **管理员邮箱**（可选）
   - 如果不填写，租户创建后需要手动创建管理员账户
   - 如果填写，系统会自动创建该邮箱对应的租户管理员账户

2. **管理员初始密码**（可选）
   - 如果不填写，默认使用：`Admin123456`
   - 如果填写，使用您输入的密码
   - **注意**：管理员首次登录后应尽快修改密码

3. **管理员姓名**（可选）
   - 如果不填写，默认使用：`租户管理员`
   - 如果填写，使用您输入的姓名

### 方式二：创建租户后手动创建管理员账户

如果创建租户时没有填写管理员信息，可以：

1. 使用平台管理员账户登录管理后台
2. 在"租户管理"中找到该租户
3. 在租户详情页面或通过其他方式创建租户管理员账户

## 示例

### 示例1：创建租户并同时创建管理员账户

```
租户名称：测试公司
管理员邮箱：admin@test-company.com
管理员初始密码：（留空，使用默认密码 Admin123456）
管理员姓名：张三
```

创建成功后：
- 租户创建成功
- 自动创建管理员账户：`admin@test-company.com`，密码：`Admin123456`

### 示例2：仅创建租户

```
租户名称：测试公司
管理员邮箱：（留空）
管理员初始密码：（留空）
管理员姓名：（留空）
```

创建成功后：
- 租户创建成功
- 需要后续手动创建管理员账户

## 注意事项

1. **默认密码**：如果未填写管理员初始密码，系统使用默认密码 `Admin123456`
2. **安全性**：管理员首次登录后应尽快修改密码
3. **邮箱唯一性**：管理员邮箱必须在系统中唯一，不能与其他用户重复
4. **自动批准**：创建租户时自动创建的管理员账户会自动通过审核，无需等待
