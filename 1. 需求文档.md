项目名称：基于DeepSeek大模型的智能简历结构化平台 MVP
核心价值：通过DeepSeek AI理解 + 可视化模板设计，实现简历智能规范化
AI核心：DeepSeek大模型承担简历理解、信息提取、智能匹配等核心任务

一、用户角色体系
1. 超级管理员
- 管理所有用户账户
- 设计和管理所有模板
- 查看平台使用统计
- 配置系统参数
2. 模板设计师
- 设计和管理简历模板
- 测试模板效果
- 维护模板版本
3. 普通会员
- 使用模板生成简历
- 管理个人简历库
- 导出Word文档
- 查看使用历史
二、核心功能模块
模块一：可视化模板设计器
- 目标：创建一个让管理员设计简历结构和格式的工具
- 功能：定义简历包含哪些字段、字段类型、验证规则、显示样式等
- 使用者：系统管理员或模板设计师
- 输出：简历模板的"蓝图"或"架构定义"
功能需求：
1. 画布设计区
  - 拖拽式组件布局
  - 实时预览效果
  - 网格对齐辅助
2. 组件库
  - 职业履历表格组件（固定：时间、公司、职位三列）
  - 经历详情面板组件（可配置字段）
  - 基础信息组件（姓名、电话、邮箱等）
  - 教育背景组件
  - 技能专长组件
3. 属性配置面板
  - 组件样式配置（字体、颜色、间距）
  - 字段显示/隐藏控制
  - 表格与详情面板关联设置
4. 模板管理
  - 模板保存/加载
  - 模板预览
  - 模板版本管理

模块二：DeepSeek驱动的智能填充系统
- 目标：将用户的原始简历内容自动填入设计好的模板中
- 功能：解析原始简历 → 匹配模板字段 → 生成规范化简历
- 使用者：普通用户
- 输出：符合模板规范的完整简历
功能需求：
1. 简历上传与预处理
  - 支持PDF、Word格式
  - 文本提取和清洗
  - 调用DeepSeek解析API
2. DeepSeek智能解析
3. 语义匹配填充
  - DeepSeek理解模板字段含义
  - 智能映射解析结果到模板字段
  - 处理字段名称不一致的情况
4. 内容优化建议
  - DeepSeek分析内容完整性
  - 建议补充关键信息
  - 优化表述专业性
模块三：Word导出功能
功能需求：
1. 格式保持
  - 保持模板设计的样式和布局
  - 表格样式完整导出
  - 字体、颜色一致性
2. 兼容性
  - 支持MS Word和WPS
  - 保持文档可编辑性
  - 分页控制

三、核心架构设计要点
1. 三层架构设计
- 设计器层：可视化操作界面，用户拖拽组件、配置属性
- 模板引擎层：将设计结果转换为可执行的模板定义
- 渲染层：根据模板定义和数据生成最终简历
2. 组件系统设计
预置组件类型：
- 表格组件：专门用于职业履历概览，固定三列配置
- 详情面板组件：与表格关联，显示选中行的详细信息
- 字段组组件：将相关字段（如汇报对象、下属团队）打包成组
- 布局容器组件：提供分栏、卡片等布局方式
组件关联机制：
- 表格组件与详情面板建立「主从关联」关系
- 当用户选择表格某行时，自动切换详情面板内容
- 两边的数据修改实时同步
3. 数据模型设计
模板元数据：
- 记录组件位置、大小、样式等布局信息
- 存储字段定义（名称、类型、验证规则）
- 保存组件间的关联关系
简历数据结构：
- 工作经历作为对象数组存储
- 每个经历对象包含表格字段和详情字段
- 支持动态添加/删除经历条目
4. 可视化操作流程
设计阶段：
1. 用户从组件库拖拽「工作经历区块」到画布
2. 系统自动创建关联的表格和详情面板
3. 用户在属性面板勾选需要显示的详情字段
4. 实时预览效果，调整样式和布局
使用阶段：
1. 用户上传简历后，系统自动填充表格数据
2. 点击表格行，在详情面板查看和编辑完整信息
3. 支持在详情面板直接添加新的工作经历
4. 关键技术实现要点
- 双向数据绑定：表格行与详情面板的数据自动同步
- 响应式布局：基于CSS Grid实现灵活的版面设计
- 字段配置系统：通过勾选方式控制详情字段的显示/隐藏
- 模板版本管理：支持模板的保存、复制和版本控制
5. 用户体验保障
- 零学习成本：拖拽操作，符合直觉
- 实时反馈：所有修改立即在预览区显示效果
- 错误预防：关键关联由系统自动处理，避免配置错误
- 灵活调整：支持随时修改字段选择和布局样式
